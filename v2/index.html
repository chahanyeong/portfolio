AI 에이전트로 일하는 방식 — 스케일 · 검증 · 도구 결합 · 학습

01 · SCALE

에이전트 팀 활용

  • - 에이전트 팀으로 기존 개발 이슈 120건의 정의·평가 수행
  • - 이 과정을 워크플로우로 구조화 — 이후 이슈 발생 시 에이전트가 자동으로 평가·적재

02 · CROSS-CHECK

GPT · Claude 교차 검토 체계

  • - GPT · Claude를 병행 활용한 교차 검토 체계 적용
  • - 단일 모델 의존으로 인한 사고 편향을 줄이기 위해 GPT · Claude 모델 혼용
  • - 두 모델 결과가 일치하면 신뢰, 불일치 시 사람이 검토 → 리뷰 비용 최소화

03 · TOOLING

플러그인 · 스킬 활용

  • - superpowers 플러그인을 기저 하네스로 활용해 작업 품질 안정화
  • - 예시: 자동 일정 캘린더 — Google Calendar에 운영 이벤트 자동 등록

04 · LEARNING

분석 데이터 자산화

  • - 분석 결과물을 자산화 — 같은 분석이 필요할 때 에이전트에게 할당
  • - 산출 결과물 공유 (예시: Mobile CCU 이상치 분석)

PROJECT 03

앱 데이터 정합성 관리

재화 정합성 76.8% 개선 · 유저 코인 미지급 건수 99% · 금액 98% 감소

서버 코인 및 유저 트랜잭션 코인 트레킹

  • · 유저 코인 정합성 오차율 76.8% 개선 (기존 88.90% → 12.10%)
  • · 고유 ID 중복 부여 유저 1,155명 식별 + 휴먼 useridx 제거로 오차율 59.18% 개선
  • · 유저 코인 미반영 예외 케이스 점검 + 로직 개선으로 17.62% 추가 개선

안드로이드 결제 이력 정합성 개선

  • · 주간 유저 코인 미지급 462개 → 5개 (99%), 미지급 금액 $16,000 → $300 (98%) 감소
  • · AOS 결제 후 구매 상품에 대응되는 재화가 들어오지 않는 문제를 이슈화
  • · Google Play Console 페이지 실시간 크롤링으로 DB에 영수증 데이터 축적
  • · 축적 영수증 ↔ 결제 테이블 대조로 불일치 영수증 건수 정량화
